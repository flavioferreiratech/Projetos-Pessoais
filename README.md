Jogo da Intuição

Um jogo simples em JavaScript que testa sua intuição 

O jogador escolhe:
Um número da seleção disponível. 
Uma cor da seleção disponível.
Uma vogal da seleção disponível.
Um signo da seleção disponível.

O jogo:
Sorteia valores aleatórios usando Math.random()
Normaliza textos para ignorar acentos e maiúsculas
Compara as escolhas do jogador
Soma a pontuação automaticamente
Exibe um feedback final baseado no resultado
